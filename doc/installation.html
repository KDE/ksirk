How to obtain KsirK

KsirK can be found at http://gna.org/projects/ksirk. There, you will find, depending of the version, a source archive (.tar.gz), a source RPM and a binary RPM made on a Mandrake Intal machine with KDE 3.4.x.

For any information, suggestion, comment or question, please mail to: (kleag AT free.fr) or use one of the KsirK mailing lists that you can find on the Web site
