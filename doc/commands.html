Chapter 3. Command Reference

The main KsirK window

There is at this time only one window and all operations are done by buttons in the toolbar. These buttons are inserted or removed when necessary. Each one have a tooltip; use it to remember yourself its usage. There is no menu nor shortcuts.

The buttons

: Quit t Game

Causes the program to imediatly quit, with no other warning ! Yes, it will be improved in a future version !

: New Game

Starts a new game. It will cause the displaying of a dialog asking the number of players. After that, each player will be asked its name with another dialog.

            : Move armies buttons

These buttons mean: invade with one, five or ten armies; stop the movement; retract one, five or ten armies. Note that one can never retract more armies than he has put.
